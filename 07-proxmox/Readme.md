## Proxmox?

Proxmox VE (Virtual Environment) is an open-source virtualization platform that allows you to run `multiple virtual machines (VMs)` and `containers` on a single physical server. It supports:

- KVM (Kernel-based Virtual Machine) for full virtualization.
- LXC (Linux Containers) for lightweight container-based virtualization.
- Web-based management interface.
- Cluster management, high availability (HA), and backup solutions.

> **Use cases:** Home labs, data centers, development environments, cloud platforms.
