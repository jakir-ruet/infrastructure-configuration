### Overview

Designed a high-availability PostgreSQL cluster using Patroni and streaming replication to ensure automatic failover and data consistency. The architecture includes one primary and multiple replicas for efficient read/write separation. It delivers fault tolerance, scalability, and minimal downtime, making it ideal for production-ready database systems.
