### Cluster Design

Designed and implemented `07-sql-ha-cluster-design`, focusing on high-availability architectures using `PostgreSQL` and `MySQL`. It covers `replication`, `automatic failover`, and `load balancing` to ensure `reliability`, `scalability`, and `minimal downtime`.

#### PostgreSQL Cluster Design – `01-postgresql`

Built PostgreSQL HA cluster using Patroni for automatic failover and replication.

- **01-manual-version:** Step-by-step manual setup with full configuration
- **02-ansible-server:** Automated deployment using Ansible playbooks

#### MySQL Cluster Design – `02-mysql`

Implemented MySQL HA cluster using Group Replication for failover and data consistency.

- **01-manual-version:** Manual configuration and setup
- **02-ansible-server:** Automated deployment using Ansible
