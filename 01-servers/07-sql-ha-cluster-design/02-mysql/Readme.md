### Overview

Designed a high-availability MySQL cluster using Group Replication, ensuring automatic failover and strong data consistency across nodes. The architecture includes one primary and multiple replicas to handle read/write workloads efficiently. It provides fault tolerance, scalability, and minimal downtime, making it suitable for production-grade database environments.
