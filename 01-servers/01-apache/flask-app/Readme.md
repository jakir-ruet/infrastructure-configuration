## Getting strated Flask Install & Apps Create

### Prepare Ubuntu & Install Python

```bash
python3 --version
pip3 --version
```

### Create a Project Folder

```bash
mkdir flask-app
cd flask-app
```

```bash
python3 -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate
```

```bash
pip install Flask
```

### Run the application

```bash
python app.py
```
